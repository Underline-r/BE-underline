## 개요

#### ➀ 언더라인?

- 방금 본 드라마의 대사, 책 속 한 줄, 누군가의 명문장에 밑줄하고 공유하는 SNS 서비스

#### ➁ 👨‍👩‍👧‍👧 팀원

- Backend: 2인 (원가연, 홍종완) [[Repo가기👉]](https://github.com/Underline-r/BE-underline)
- Frontend: 2인 (정성목, 최경민) [[Repo가기👉]](https://github.com/Underline-r/FE-underline_web)

#### ➂ 개발 기간 : 2022.08.01 ~

#### ➃ 기술 스텍

- Java
- MySQL
- SpringBoot
- QueryDSL
  

<br/>

## 프로젝트 주요 기능

#### ➀ 기능 요약
- 게시글 CRUD 및 페이징
- 댓글/해시태그/좋아요
- 유저 정보 조회
- 팔로우/팔로잉

#### ➁ 신경쓴 기능
- 메일 서버
- 조회수 배치
- 검색 기능
- 쿼리 DSL을 통한 타임라인 조회
- 공통 예외처리
